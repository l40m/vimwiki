SSH

Dependencias

  • openssh-server

Configuración

  1. Elige una contraseña con passwd
  2. Inicia el sistema con > sudo service ssh start

Si quieres que se mantenga a cada inicio de sesión puede añadirlo a tu $HOME/.xinitrc como: sudo service ssh start &

backup y restore de Tiny Core Linux

Añadir estas lineas a /opt/.filetool.lst: etc/ssh etc/shadow
