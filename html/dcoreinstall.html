Introducción

dCore Linux es una versión de la distribución GNU/Linux, Tiny Core Linux. Es ligera, modular y estable. Se comporta como un sistema empotrado. A diferencia del Tiny Core Linux, si bien ambos suplen sus necesidades con extensiones, es decir, programas especiales para la arquitectura.

Instalador

Bootloader

Esta son las lineas que uso actualmente.

noautologin lang=es_MX.UTF-8 tce=sda5 opt=sda5 home=sda5 laptop desktop=i3 tce=sda1 opt=sda1 home=sda1 laptop kmap=qwerty/es lang=es_MX.UTF-8 tz=UTC-8 noautologin multivt video=SVIDEO-1:d desktop=i3
  1. noautologin: No se inicia sesión automáticamente. Solicita una contraseña para iniciar sesión (se requiere si se pretende usar ssh)
  2. lang: Lenguaje.
  3. tce: Directorio Tiny Core Linux.
  4. opt: Directorio Tiny Core Linux para configuraciones.
  5. home: Directorio personal.
  6. desktop: Nombre de la aplicación usada como entorno de escritorio.
