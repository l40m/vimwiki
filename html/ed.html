Estadística descriptiva

Calcular la media, mediana y la moda

Con statistics:

  • Para la media:

print (statistics.mean([a,a,a,a]))

  • Para la mediana:

print (statistics.median([a,a,a,a]))

  • Para la moda:

print (statistics.mode([a,a,a,a,a]))

Calcular la media ponderada:

Con numpy: import numpy as np print (np.average([1, 2, 3], weights=[1, 0, 1]))

Con wieightedstats: import weightedstats as ws

my_data = [1, 2, 3, 4, 5] my_weights = [10, 1, 1, 1, 9]

ws.weighted_mean([a,a,a,a], weights=[a,a,a,a]) ws.weighted_median([a,a,a,a], weights=[a,a,a,a])

Media y mediana ponderada especial, función para ser usada con numpy array ws.numpy_weighted_mean([], weights=[]) ws.numpy_weighted_median([], weights=[])
