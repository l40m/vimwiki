Ruby

Compilar ruby

  • Descarga la última versión del sitio oficial
  • Carga la extensión para compilar, seguramente compile
  • Descomprime el archivo, situate en la carpeta
  • Ejecuta: ./configure --prefix=$HOME/.local, make y finalmente sudo make install.

Compila el soporte para openssl

> Para instalar gemas usando https requieres soporte de openssl.

  • Entra a la carpeta ext/openssl y compila usando ruby extconf.rb
  • Edita el Makefile añadiendo top_srcdir = ../.. abajo de archdir.
  • Ejecuta make y sudo make install.

Problemas frecuentes

ERROR: Could not find a valid gem 'jekyll' (>= 0)

Lo anterior se soluciona agregando: gem sources --remove https://rubygems.org/ y despues; gem sources -a http://rubygems.org/.
