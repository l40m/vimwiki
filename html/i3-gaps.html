i3-gaps

Dependencias

Para compilar, i3-gaps requiere de ciertas dependencias. Algunas generales, necesarias para compilar cualquier otro paquete. La página del proyecto tienes una lista de dependencias más adecuada según la distribución que se este usando.

Para usuarios de dCore Linux: puedes hacer un archivo de texto llamada i3-gaps con todas estas extensiones e importarlas como un meta-paquete: sce-import -l i3-gaps. La libreria xcb se obtiene del siguiente ppa: ppa:aguignard/ppa. La extensiones siguientes son para dCore Xenial: pkg-config make automake autoconf gcc git libpng12-dev libxcb-shape0-dev libxcb1-dev libxcb-keysyms1-dev libpango1.0-dev libxcb-util0-dev libxcb-icccm4-dev libyajl-dev libstartup-notification0-dev libxcb-randr0-dev libev-dev libxcb-cursor-dev libxcb-xinerama0-dev libxcb-xkb-dev libxkbcommon-dev libxkbcommon-x11-dev autoconf libxcb-xrm-dev

Instalación

  • Satisface las dependencias
  • Clona el repositorio y sitúate dentro de el: git clone https://www.github.com/Airblader/i3 i3-gaps && cd i3-gaps
  • Compila e instala: autoreconf --force --install
  • Esto ni idea qué hace, pero idk: rm -rf build/ && mkdir -p build && cd build/
  • Configura: ./configure --prefix=~/.local --sysconfdir=~/.local --disable-sanitizers
  • Compila: make && sudo make install
