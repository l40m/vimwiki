Pandoc

Instalación

  • Importa y carga la extensión haskell-platform para conseguir los binarios de cabal haciendo cabal update.
  • Carga ghc, que es dependencia.
  • Instala pandoc usando cabal install pandoc.
  • stack update -j1 +RTS -M900M -RTS
  • stack upgrade -j1 +RTS -M900M -RTS
  • cd pandoc
  • stack install -j1 +RTS -M900M -RTS
