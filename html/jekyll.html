Blog con Jekyll de Ruby y GitHub Pages de ya saben quien

  1. Jekyll necesita la versión 2.4.0 de Ruby, así que necesitas compilarlo manualmente. Necesitas cargar make, pkg-config, libssl-dev y g++ (estos paquetes los contiene la meta extensión de usuario compile).
  2. Para mayor soporte, necesitarás cargar libreadline-dev, libssl-dev y zlib1g-dev.
  3. Es posible que requieras especificar la localización de esas librerias. Por lo que el comando que sigue deberia bastar:
  4. ./configure --prefix=/tmp/ruby-2.7.1 --with-openssl-dir=/usr/local/ssl --with-readline-dir=/usr --with-zlib-dir=/usr

Crear un sitio web estático

gem install bundler jekyll jekyll new nombre cd nombre bundle exec jekyll serve http://localhost:4000
